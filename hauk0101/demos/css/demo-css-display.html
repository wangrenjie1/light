子元素 display 的 inline、inline-block、block 对 text-align 的影响

统一父元素为 display:block ，text-align:center 的设置

子元素设置 display:block
display:block 子元素

子元素是block,则子元素的文字相对子元素居中，而子元素并不会相对父元素居中

子元素设置 display:inline-block
display:inline-block 子元素

子元素是inline-block，则子元素的文字不仅相对子元素居中，而且子元素相对父元素也居中

子元素设置 display:inline
display:inline 子元素

子元素是inline，则子元素的文字无法撑起子元素，子元素的宽高就是文字的宽高，而子元素是相对父元素居中的（设置宽高并不会生效）
